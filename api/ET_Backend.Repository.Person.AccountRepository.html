Class AccountRepository

Repository für den Zugriff auf Accounts / Users – funktioniert sowohl mit SQLite (Dev) als auch mit Azure SQL (Prod). Plattformabhängige Details (Tabellennamen, Identity‑Abruf) werden zur Laufzeit ermittelt.
