GetBase64FromImage(string)

Liest eine Bilddatei vom angegebenen relativen Pfad (bezogen auf das aktuelle Arbeitsverzeichnis) und konvertiert deren Inhalt in einen Base64-codierten String. Diese Methode wird z. B. beim Initial-Seeding von Bilddaten (z. B. Logos) verwendet.

Declaration
public static string GetBase64FromImage(string relativePath)
Parameters
Type Name Description
string relativePath

Relativer Pfad zur Bilddatei, z. B. "Resources/Seed/BitWorksSimpel-Gro.png". Der Pfad wird relativ zu Directory.GetCurrentDirectory() aufgelöst.
Returns
Type Description
string

Ein Base64-codierter String des Bildes – ohne "data:image/...;base64,"-Präfix. Gibt null zurück, wenn die Datei nicht gefunden oder nicht gelesen werden konnte.
